These files are kept for reference but should not be used. Use the scripts in `/scripts` instead.

- `test_balance_check.py` - Balance verification prototype
- `check_kucoin_mm2.py` - KuCoin MM2 account testing

- `check_cache_stats.py` - Cache statistics (moved to scripts/cache_stats.py)
- `analyze_price_impact.py` - Price impact analysis (moved to scripts/price_impact.py)
